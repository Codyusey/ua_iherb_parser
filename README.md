# ua_iherb_parser
___
### Description:
- One-thread parser  'https://ua.iherb.com/'
- Parse only from ukrainian IP 
- Parsing speed - 3 items / min. approx
### Minimum system requirements:
- OS: Windows 10 or 11
- Python 3.9.7 or latest
