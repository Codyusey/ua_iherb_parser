# ua_iherb_parser
## Description:
- One-thread parser based on Selenium undetected chrome  
- Target: 'https://ua.iherb.com/'
- Parse only from ukrainian IP 
- Parsing speed approximately - 3 items/min. 
- ## System requirements:
- Windows 10 (64 bit) or Windows 11
- Python 3.9.7 or latest
